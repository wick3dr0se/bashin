<details open><summary><h1>:heavy_division_sign: Math</h1></summary>

A math library, written in C, capabale of handling advanced floating point arithmetic in precedence

### Arithmetic
Evaluate a mathematical equation
```bash
math <equation>
```

Example:
```bash
math 2.2+5
```
> 7.2

#### Supported operators
```
+  ...  sum
-  ...  difference
*  ...  product
/  ...  quotient
%  ...  remainder
```
